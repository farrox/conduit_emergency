Getting Started

Installation

Before you begin, ensure you have Go 1.24.x installed. See the Setup Guide for detailed installation instructions.

First Time Setup

# Clone the repository (if not already done)
git clone [repository-url]
cd conduit_emergency

# Run initial setup (clones required dependencies)
make setup

Build the Project

# Build for your current platform
make build

# The binary will be in dist/conduit

Configuration

You'll need a Psiphon network configuration file. Contact Psiphon (info@psiphon.ca) to obtain valid configuration values.

See psiphon_config.example.json in the project root for the expected format.

Run Your First Node

# Start with default settings
./dist/conduit start --psiphon-config ./psiphon_config.json

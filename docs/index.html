Conduit CLI Documentation

Command-line interface for running a Psiphon Conduit node - a volunteer-run proxy that relays traffic for users in censored regions.

New to this? Start with the Quick Start Guide for Mac users.

Quick Start

# First time setup (clones required dependencies)
make setup

# Build
make build

# Run
./dist/conduit start --psiphon-config /path/to/psiphon_config.json

Requirements

  • Go 1.24.x (Go 1.25+ is not supported due to psiphon-tls compatibility)
  • Psiphon network configuration file (JSON)

See Setup Guide for installation instructions.

Easy Installation for macOS

For Intel-based Mac users, we've made installation as simple as possible:

  • DMG Installer: Download, drag to Applications, and run
  • Automated Setup: One script handles everything for cloned repos
  • Simple Launcher: Double-click to start

See the Installation Guide for details.

Optimal Configuration

Configure optimal settings for maximum users with your bandwidth:

# Interactive configuration helper
./scripts/configure-optimal.sh

# Or quick start with auto-detection
./scripts/quick-optimal.sh

See CONFIG_OPTIMAL.md for detailed guidance.

Features

  • Volunteer-run proxy for users in censored regions
  • Configurable client limits and bandwidth
  • Docker support with embedded configuration
  • Cross-platform builds
  • Easy macOS installation (DMG + automated setup)

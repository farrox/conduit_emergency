Step 2: Get Your Config File

The config file connects you to the Psiphon network. No email needed if you have the iOS app!

If You Have the iOS Conduit App:

./scripts/extract-ios-config.sh

That's it! The script extracts the config from your iOS app.

If You Don't Have the iOS App:

Email info@psiphon.ca with subject "Request for Conduit CLI Configuration"

More details on getting your config â†’

Step 3: Start Conduit

You're ready! How you start depends on your installation method:

  • Docker Manager: Just run the script - it handles everything!
  • DMG: Double-click "Start Conduit.command" in Applications
  • From Source: Double-click "Start Conduit.command" in the project folder

What is Conduit?

Conduit is a volunteer-run proxy service that helps Iranians access the open internet. By running a Conduit node, you're providing a valuable service to help Iranians bypass internet censorship.

Live Dashboard

Monitor your Conduit node in real-time with the built-in dashboard. See connected Iranians, CPU/RAM usage, and traffic statistics.

Live Dashboard showing real-time monitoring

To use the dashboard:

# Start Conduit with stats enabled
./dist/conduit start --psiphon-config ./psiphon_config.json -v --stats-file

# In another terminal, run the dashboard
./scripts/dashboard.sh

Or use the convenience script that starts both:

./scripts/start-with-dashboard.sh
